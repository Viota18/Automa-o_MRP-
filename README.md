# Automa-o_MRP-
Este projeto demonstra uma automação desenvolvida em Python para processamento de dados de planejamento de materiais (MRP). O sistema realiza leitura de planilhas, tratamento de dados, validações e geração de saídas estruturadas.
# Automação de Processos de Compras com Python (MRP)

> ⚠️ Observação: Este projeto é uma versão adaptada para portfólio, sem dados ou informações confidenciais.

---

## Funcionalidades

* Leitura de arquivos Excel
* Tratamento de dados com pandas
* Validações (datas, quantidade, lead time)
* Geração de logs de erros e alertas
* Exportação de resultados

---

## Tecnologias utilizadas

* Python
* pandas
* openpyxl

##  Como executar

1. Instale as dependências:

```
pip install pandas openpyxl
```

2. Execute o script:

```
python automation.py
```

---

##  Exemplo de uso

O sistema lê uma planilha contendo dados de materiais e realiza:

* Verificação de datas inválidas
* Validação de quantidade
* Cálculo e organização de dados
